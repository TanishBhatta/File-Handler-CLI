# File Handler CLI

A Python tool that lets you navigate your file 
system and perform full CRUD operations on any file  
without leaving the terminal.

---

## How to Run manually

```bash
git clone https://github.com/TanishBhatta/file-handler-cli
cd file-handler-cli
pip install colorama
python file_handler.py
```

---

## Functions

- **Create** — make a new file in any folder with optional 
  text written inside it on creation
- **Read** — open and display the contents of any existing file
- **Update** — rename a file, overwrite its content, 
  or append new content
- **Delete** — permanently remove a file from disk

---

## Requirements (not required if installing .exe)

- Python 3.x
- colorama
- tkinter (included in standard Python installation)
